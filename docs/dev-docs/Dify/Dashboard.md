The dashboard tracks four metrics over time to show how your application performs:![monitoring-dashboard.png](https://mintcdn.com/dify-6c0370d8/MiGqQWUuc2Ca3g05/images/monitoring-dashboard.png?fit=max&auto=format&n=MiGqQWUuc2Ca3g05&q=85&s=f71e104219b15e5a775c891db7037fe3 "monitoring-dashboard.png") **Total Messages:** Conversation volume  
**Active Users:** Users with meaningful interactions (more than one exchange)  
**Average User Interactions:** Engagement depth per session  
**Token Usage:** Resource consumption and costs Use the time selector to view trends over different periods. Click **“Tracing app performance”** to connect external observability platforms like Langfuse or LangSmith for deeper analytics.
